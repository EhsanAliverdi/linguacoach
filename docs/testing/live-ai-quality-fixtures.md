# Live AI Quality Fixtures

Use these sample answers during manual live AI review. They are synthetic and contain no real user data.

## Fixture 1: Direct Tone Follow-Up

Use for Project Planner or Document Controller follow-up activities.

Expected issues:

- Direct tone
- Missing softening language
- Slightly unclear request

```text
Hi John,

I sent the schedule last week and you still did not approve it. Please approve today because I need to finish my work.

Thanks
```

Good improved direction:

```text
Hi John,

I wanted to follow up on the schedule I sent last week. Could you please review it when you have a chance? I need the approval before I can finalise the next project update.

Thank you,
Sara
```

## Fixture 2: Long Unclear Progress Update

Use for Project Planner progress update activities.

Expected issues:

- Long sentence
- Weak summarising
- Unclear next step
- Article/preposition mistakes

```text
Dear Manager,

I am writing about programme update and because supplier send information late and design team still checking the drawing so I think maybe we can finish report later but I am not sure and I will tell you when it is done.

Regards
```

Good improved direction:

```text
Dear Manager,

I am writing to update you on the programme report. The supplier sent the information late, and the design team is still checking the drawing. I expect to send the revised report by Thursday.

Regards,
Sara
```

## Fixture 3: Missing Articles And Tense Issue

Use for explanation-of-delay activities.

Expected issues:

- Missing articles
- Tense issue
- Vague timing
- Needs accountability

```text
Dear Mr Lee,

Report is delay because supplier did not sent final file. I will send when I receive it.

Best regards
```

Good improved direction:

```text
Dear Mr Lee,

The report is delayed because the supplier has not sent the final file yet. I apologise for the delay and will send the report by Friday if the file arrives today.

Best regards,
Sara
```

## Fixture 4: Overly Casual Customer Reply

Use for Customer Support Officer complaint-response activities.

Expected issues:

- Too informal
- Low empathy
- Needs clear explanation
- Needs a professional next step

```text
Hi,

Sorry about that. We are busy now and cannot fix it today. You can wait until tomorrow and maybe it will be ok.

Thanks
```

Good improved direction:

```text
Hello,

I am sorry for the inconvenience. Our team is currently reviewing the issue, and we expect to provide an update by tomorrow morning. Thank you for your patience.

Kind regards,
Sara
```

## Fixture 5: Stronger Second Attempt

Use after feedback to check score comparison and whether the AI recognises improvement.

Expected result:

- Higher score than direct first attempt
- Fewer tone changes
- Feedback should still suggest one practical next improvement

```text
Dear John,

I wanted to follow up on the schedule I sent last week. Could you please review it when you have time? Your approval is needed before I can finalise the next project update.

Best regards,
Sara
```

## Review Notes

When using these fixtures, record:

- Activity title and module.
- Score.
- Top feedback categories.
- Whether before/after changes referenced exact text.
- Whether the mini lesson matched the biggest issue.
- Whether Persian explanation stayed concise.
- Whether memory updated in a believable way.
