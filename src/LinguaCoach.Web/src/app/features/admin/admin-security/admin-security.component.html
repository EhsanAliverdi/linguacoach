These capabilities are planned or partially implemented. None are active in the current build.

Multi-factor authentication (MFA/TOTP)
Deferred
TOTP/app-based MFA is not yet wired. No enrolment, challenge, or recovery flows exist.
Enterprise SSO — SAML / OIDC
Deferred
External identity provider federation is not implemented. Google OAuth scaffold exists but full OIDC/SAML is deferred.
Distributed / Redis rate limiting
Deferred
Current rate limiting is in-process memory only. Multi-instance deployments need a Redis-backed store.
Captcha / abuse protection
Deferred
No CAPTCHA or bot-mitigation on login, registration, or password reset flows.
CSP / HSTS header hardening
Deployment dependent
Content-Security-Policy and HSTS enforcement should be configured at the reverse-proxy or hosting layer. Not enforced by the application in development.
SMS security notifications
Deferred
Login alerts and password-change SMS notifications require both an SMS provider and phone verification, neither of which is implemented.
Admin session / device management UI
Planned
No UI exists for admins to view or revoke active sessions or trusted devices.
